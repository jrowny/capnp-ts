# capnp-ts

this is a fork of [jdiaz5513/capnp-ts](https://github.com/jdiaz5513/capnp-ts/) which works with Typescript 5

## setup

clone and make sure you have `capnpc` in your path. This is available in most package managers like `brew install capnpc`.

**Important:**, if you're generating other languages, make sure you use the same version.
